## 系统概述

本系统通过**5轮独立分析**对上市公司进行全面研究，每轮使用6个专业代理，共产生30份独立分析报告，最终通过交叉验证生成经过校验的投资研究报告。

### 核心特点

- ✅ **自动化流程**：从下载财报到生成报告全程自动化
- ✅ **多轮验证**：5轮独立分析确保数据准确性
- ✅ **专业分工**：6个代理各司其职，深度分析
- ✅ **交叉验证**：自动对比并解决数据冲突
- ✅ **双语输出**：自动生成中英文版本
- ✅ **图表丰富**：每份报告包含专业数据图表

### 分析代理

| 代理名称 | 分析内容 | 输出报告 |
|---------|---------|---------|
| **segment-agent** | 业务构成与成本结构 | 业务板块收入、地理分布、成本结构 |
| **balance-agent** | 资产负债表 | 资产质量、债务水平、财务健康度 |
| **value-agent** | 企业估值 | DCF模型、内在价值、安全边际 |
| **cap-agent** | 资本配置 | 分红政策、股票回购、投资效率 |
| **cap-acq-agent** | 投资入股与并购 | 对外投资、并购交易历史 |
| **mda-agent** | 管理层讨论与分析 | 高管团队、战略观点、可信度评分 |

## 快速开始

### 方法一：使用Claude Code（推荐）

1. **打开Claude Code**
   ```bash
   # 在项目目录中启动
   cd claude-invest
   ```

2. **执行分析命令**
   ```
   /研究 Apple Inc.
   ```

3. **等待分析完成**
   - 系统会自动：下载财报 → 5轮分析 → 交叉验证 → 生成报告
   - 全程约需30-60分钟

4. **查看结果**
   ```bash
   # 查看最终报告
   cat input/*投资研究报告.md

   # 查看README更新
   cat README.md
   ```

### 方法二：使用自动化脚本

```bash
# 美股公司
python auto_analysis.py "Apple Inc." SEC

# 港股公司
python auto_analysis.py "腾讯控股" HKEX

# 或使用shell脚本
bash run_analysis.sh "Apple Inc." SEC
```

## 详细流程

### 步骤1：准备财报数据

**自动化方式**：
```
使用 prepare-agent 或 sec-fetcher/hkex-fetcher skill
```

**手动方式**：
1. 访问SEC EDGAR（美股）或HKEXnews（港股）
2. 下载最近5年的年报：
   - 美股：10-K（本土公司）或20-F（外国公司）
   - 港股：年度报告
3. 保存到 `input/` 目录
4. 使用 stockanalysis skill 获取财务数据
5. 创建 `input/references.md` 文件

**references.md 格式示例**：

```markdown
# 财务数据参考

## 利润表（最近5年）

|------|------------|----------|--------|
| 2024 | $206,360   | $3,889   | $2,869 |
| 2023 | $205,980   | $4,078   | $3,075 |
...

## 资产负债表

...

## 现金流量表

...

## 分红数据

| 年份 | 每股分红 | 派息率 |
|------|----------|--------|
| 2024 | $2.50    | 50.5%  |
...
```

### 步骤2：执行5轮分析

**在Claude Code中**：

```
/研究 [公司名称]
```

系统会自动执行：

```markdown
【第1轮分析】
1. segment-agent → 业务构成与成本结构分析报告
2. balance-agent → 资产负债表分析报告
3. value-agent → 企业估值分析报告
4. cap-agent → 资本配置分析报告
5. cap-acq-agent → 投资入股与并购分析报告
6. mda-agent → MD&A管理层讨论分析报告

【第2轮分析】
...（重复以上6个代理）

【第3轮分析】
...

【第4轮分析】
...

【第5轮分析】
...
```

**每轮输出文件**：

```
input/
├── [公司名]_业务构成与成本结构分析报告_第N轮_YYYY-MM-DD.md
├── [公司名]_资产负债表分析报告_第N轮_YYYY-MM-DD.md
├── [公司名]_独立估值分析报告_第N轮_YYYY-MM-DD.md
├── [公司名]_资本配置分析报告_第N轮_YYYY-MM-DD.md
├── [公司名]_投资入股与并购研究报告_第N轮_YYYY-MM-DD.md
└── [公司名]_MD&A独立分析报告_第N轮_YYYY-MM-DD.md
```

### 步骤3：交叉验证

使用 **diff-agent** 对每个模块的5轮报告进行比对：

```markdown
对每个模块：
1. 读取5轮报告
2. 提取所有数据点
3. 比对一致性
4. 解决冲突（多数原则、来源优先级）
5. 生成校验报告
```

**输出文件**：

```
input/
├── 业务构成与成本结构分析_最终校验报告.md
├── 资产负债表分析_最终校验报告.md
├── 企业估值分析_最终校验报告.md
├── 资本配置分析_最终校验报告.md
├── 投资入股与并购分析_最终校验报告.md
└── MD&A管理层讨论分析_最终校验报告.md
```

### 步骤4：生成最终报告

基于校验报告生成最终投资研究报告：

```
input/[公司名称]_投资研究报告.md
```

**报告结构**：

```markdown

## 报告摘要
```

## 常见问题

### Q1: 分析需要多长时间？

**A**: 完整流程约需30-60分钟：
- 下载财报：5-10分钟
- 5轮分析（30份报告）：20-40分钟
- 交叉验证：5-10分钟
- 生成最终报告：2-5分钟

### Q2: 可以减少分析轮次吗？

**A**: 可以，但不建议。系统设计5轮是为了：
- 确保数据准确性（避免单轮错误）
- 识别数据冲突
- 提高可信度

如需修改，编辑 `auto_analysis.py`：
```python
self.rounds = 3  # 改为3轮
```

### Q3: 如何分析港股公司？

**A**:
1. 确保安装 `agent-browser`（用于HKEX网站交互）
2. 使用 `hkex-fetcher` skill 下载年报
3. 其余流程与美股相同

### Q4: 报告中的数据不一致怎么办？

**A**: 系统会自动处理：
- **5/5一致**：直接采用
- **4/5一致**：采用多数值，报告中记录
- **3/5一致**：标注需关注
- **2/5或更少**：用红色标识，建议校验

所有差异都会在"数据校验说明"章节中详细说明。

### Q5: 如何自定义分析模块？

**A**:
1. 在 `.claude/skills/` 创建新skill
2. 在 `.claude/agents/` 创建对应agent
3. 在 `invest-report/SKILL.md` 中添加到分析流程
4. 更新报告模板

### Q6: 图表无法显示怎么办？

**A**:
1. 确保 `generated_images/` 目录存在
2. 检查Python环境和matplotlib安装
3. 验证图表文件路径正确

### Q7: 如何验证报告质量？

**A**: 检查以下指标：
1. **数据一致性**：查看"数据校验说明"章节
2. **逻辑连贯性**：各部分分析应相互支撑
3. **图表完整性**：应该有6-10张图表
4. **结论明确性**：应该有明确的买入/持有/卖出建议

## 高级配置

### 自定义数据源

编辑 `.claude/skills/invest-report/references/common-guidelines.md`：

```markdown
### 数据来源评估标准

| 来源 | 权威性 | 严谨性 | 相关性 |
|-----|-------|-------|-------|
| 年报官方数据 | 高 | 高 | 高 |
| stockanalysis.com | 中 | 高 | 高 |
| Bloomberg | 高 | 高 | 高 |  # 添加新来源
```

### 修改估值参数

编辑 `.claude/skills/invest-value/SKILL.md`：

```markdown
**贴现率设置**：
- 基准：10%
- 悲观：12%
- 乐观：8%

**增长率假设**：
- 悲观：-5%
- 基准：5%
- 乐观：7%
```

### 添加新交易所

1. 创建新的fetcher skill：
   ```
   .claude/skills/[exchange]-fetcher/SKILL.md
   ```

2. 更新 prepare-agent：
   ```markdown
   判断公司在哪上市，不确定的问我
   美股：使用 sec-fetcher skill
   港股：使用 hkex-fetcher skill
   [新交易所]：使用 [新exchange]-fetcher skill
   ```

### 批量分析

创建批量分析脚本：

```bash
#!/bin/bash
# batch_analysis.sh

COMPANIES=(
    "Apple Inc."
    "Microsoft Corporation"
    "Alphabet Inc."
)

for COMPANY in "${COMPANIES[@]}"; do
    echo "Analyzing $COMPANY..."
    python auto_analysis.py "$COMPANY" SEC
    echo "Completed $COMPANY"
    echo "---"
done
```
