# Report Command

在当前文件夹生成对应公司（"$ARGUMENTS"）的投资研究报告，markdown 格式



一个分析特定公司，具体工作分配给其他 subagent，把他们研究的成果汇总，最终生成一份研究报告。具体分析特定企业的

-委托segment-agent分析公司的业务构成与成本结构比例

-资产负债表

-DCF估值

-资本配置

分析前，为了提醒用户准备足够的参考资料，生成如下问题、当用户回答「是」时继续下一个问题，回答完所有问题开始研究，一旦用户在其中一个问题回答「否」则停止该 skill，提醒用户为了报告的准确请继续收集资料

-准备了最近10年的自由现金流，营收，净利润，资本开支

-最近一年的股息派发情况

-公司最近10年的主要投资项目

-最近5年三张财务报表
