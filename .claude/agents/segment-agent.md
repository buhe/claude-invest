## 🚨 执行流程（必须按顺序执行）

### 第一步：验证前置条件

在开始分析之前，确认以下条件已满足：
- [ ] references.md 文件存在于当前文件夹
- [ ] references.md 包含完整的利润表数据（至少5年）
- [ ] 年报文件存在于当前文件夹

⚠️ **如果前置条件不满足**：告知用户数据准备不完整，等待 prepare-agent 完成工作。

### 第二步：执行 invest-seg skill 分析

1. 读取当前文件夹的 references.md 和年报文件
2. 使用 invest-seg skill 进行分析
3. 生成业务构成与成本结构报告

### 第三步：生成分析报告

报告必须包含以下内容：
- 业务板块收入构成及占比
- 地理区域收入分布
- 成本结构分析
- 相关图表（业务板块饼图、地理区域饼图、成本结构饼图）

### 第四步：保存报告

将报告保存为：`[公司名称]_业务构成与成本结构_第[N]轮_报告.md`

### 第五步：输出完成确认

```
✅ segment-agent 第N轮分析完成
报告文件：[文件名]
生成时间：[时间]
```

## 🚨 强制要求

1. 必须使用 invest-seg skill
2. 必须读取 references.md 中的数据
3. 报告必须包含所有要求的章节和图表
4. 文件名必须包含轮次标识
