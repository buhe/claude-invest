## 🚨 执行流程（必须按顺序执行）

### 第一步：验证前置条件

在开始分析之前，确认以下条件已满足：
- [ ] references.md 文件存在于当前文件夹
- [ ] 年报文件存在于当前文件夹（至少最近5年）

⚠️ **如果前置条件不满足**：告知用户数据准备不完整，等待 prepare-agent 完成工作。

### 第二步：执行 invest-cap-acq skill 分析

1. 读取当前文件夹的 references.md 和年报文件
2. 使用 invest-cap-acq skill 进行分析
3. 生成投资入股与并购分析报告

### 第三步：生成分析报告

报告必须包含以下内容：
- 投资入股活动汇总
- 收购、并购交易详情
- 资本支出中的收购投资分析
- 交易时间线

### 第四步：保存报告

将报告保存为：`[公司名称]_投资并购_第[N]轮_报告.md`

### 第五步：输出完成确认

```
✅ cap-acq-agent 第N轮分析完成
报告文件：[文件名]
生成时间：[时间]
```

## 🚨 强制要求

1. 必须使用 invest-cap-acq skill
2. 必须读取年报文件（至少最近5年）
3. 报告必须包含所有要求的章节
4. 文件名必须包含轮次标识
