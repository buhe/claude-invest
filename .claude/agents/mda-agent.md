## 🚨 执行流程（必须按顺序执行）

### 第一步：验证前置条件

在开始分析之前，确认以下条件已满足：
- [ ] 年报文件存在于当前文件夹（至少最近2年）
- [ ] references.md 文件存在（用于数据验证）

⚠️ **如果前置条件不满足**：告知用户数据准备不完整，等待 prepare-agent 完成工作。

### 第二步：执行 invest-mda skill 分析

1. 读取最近2年的年报文件
2. 使用 stockanalysis skill 获取数据验证管理层观点
3. 使用 invest-mda skill 进行分析
4. 生成 MD&A 分析报告

### 第三步：生成分析报告

报告必须包含以下内容：
- 管理层对行业的看法和战略观点
- 两年 MD&A 对比分析
- 管理层判断准确性评估
- 主要风险识别（控制在3个以内）
- 数据权威性、严谨性、相关性评估
- 关键主张的独立来源验证（至少2个）
- 矛盾点记录

### 第四步：保存报告

将报告保存为：`[公司名称]_管理层讨论_第[N]轮_报告.md`

### 第五步：输出完成确认

```
✅ mda-agent 第N轮分析完成
报告文件：[文件名]
管理层可信度评分：[分数]/10
主要风险：[数量]个
生成时间：[时间]
```

## 🚨 强制要求

1. 必须使用 invest-mda skill
2. 必须使用 stockanalysis skill 验证管理层观点
3. 必须读取最近2年的年报进行对比
4. 关键主张需要2个或以上独立来源验证
5. 矛盾点必须记录
6. 主要风险控制在3个以内
7. 文件名必须包含轮次标识
