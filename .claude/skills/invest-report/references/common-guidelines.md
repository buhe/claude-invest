# 投资分析公共规范

本文档包含所有投资分析 skill 的公共规范要求。各 skill 应引用本文档而非重复这些内容。

## 一、数据验证通用规范

### 数据来源评估标准

| 来源 | 权威性 | 严谨性 | 相关性 |
|-----|-------|-------|-------|
| 年报官方数据 | 高 | 高 | 高 |
| stockanalysis.com | 中 | 高 | 高 |
| 参考资料 | 评估 | 评估 | 评估 |
| 其他来源 | 评估 | 评估 | 评估 |

### 验证要求

1. **交叉验证**：关键数据至少从2个独立来源确认
2. **相同出处=1个来源**：引用相同出处的来源算作1个来源（非2个）
3. **记录矛盾**：透明记录所有矛盾点，禁止隐藏
4. **单位统一**：确保所有数值使用相同单位
5. **逻辑检查**：资产=负债+权益，流动+非流动=总计

### 使用 stockanalysis skill

```
使用 stockanalysis skill 获取最近五年的财务数据进行交叉验证
```

## 二、图表生成通用规范

### 关键要求

- **禁止AI生成**：使用 Python matplotlib 生成真实数据图表
- **英文标签**：所有图表标签、标题和文本必须使用英文
- **分辨率**：300 DPI
- **保存路径**：`generated_images/` 子目录
- **年份提取**：从实际数据中动态提取年份，禁止硬编码
- **数量限制**：每份报告最多 10 张图表

### 图表尺寸规范

| 图表类型 | 尺寸 | 宽高比 |
|---------|------|--------|
| 折线图 | 12x6英寸 | 16:9 |
| 柱状图 | 12x6英寸 | 16:9 |
| 饼图 | 10x8英寸 | 1:1 |

### 图片文件名规范

```
业务板块:     business_segments.png
地理区域:     geographic_segments.png
成本结构:     cost_structure.png
资产构成:     asset_composition.png
负债构成:     liability_composition.png
债务趋势:     debt_trend.png
收入趋势:     revenue_trend.png
净利润趋势:   net_income_trend.png
现金流趋势:   fcf_trend.png
股息增长:     dividend_growth.png
股数趋势:     share_count_trend.png
资本配置:     capital_allocation_breakdown.png
```

### 图表插入格式

```markdown
![图表描述](generated_images/图表文件名.png)
```

## 三、报告语言规范

- **报告正文**：中文
- **图表标签/标题**：英文
- **专业术语**：使用标准中文金融术语

## 四、文件命名规范

- 报告文件：`[公司名称]_[分析类型]_YYYY-MM-DD.md`
- 日期使用昨日日期作为参考点

## 五、财报使用原则

**只使用最新一年的财报作为分析资料**，以保证专注度。

对于 MD&A 分析等需要历史对比的场景，额外读取前一年财报。
