## 使用说明

### 更新规则库
规则库基于书本原文抽取，每条规则关联 `source_chapter` 和 `reference_quote`，确保可溯源。

- 财报疑点排查（唐门排雷六步法）
- 财务造假识别
- A股/港股上市公司财报分析
- 价值投资尽调辅助
